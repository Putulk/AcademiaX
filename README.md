# AcademiaX

A cloud-native ERP Management System for Schools and Colleges built using Java Microservices.

## Features

- Role-Based Authentication & Authorization (RBAC)
- Student Management
- Teacher Management
- Course Management
- Attendance Management
- Examination & Results
- Fee Management
- HR Management
- Library Management
- Notification Service
- Reporting & Analytics

---

- Java 21
